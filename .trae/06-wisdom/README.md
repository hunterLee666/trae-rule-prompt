# 06-wisdom/ 智慧框架层

## 职责范围

智慧框架层是TRAE系统的"灵魂"，负责：
- **智慧框架** - 53个思维框架的整合应用
- **持续学习** - 从交互中学习和进化
- **系统进化** - 架构和能力的持续改进
- **超越智慧** - 深度洞察和范式突破

---

## 核心组件

### 1. 智慧框架 (wisdom-frameworks.md)
53个框架的深度整合：

```yaml
框架分类:
  Eastern (东方智慧 - 9框架):
    - 道家无为
    - 禅宗顿悟
    - 不二整体观
    - 空性之舞
    - ...
    
  Science (科学智慧 - 8框架):
    - 复杂性智能
    - 量子认知
    - 集体智能
    - 系统思维
    - ...
    
  Human (人类智慧 - 8框架):
    - 七美德
    - 意义建构
    - 爱的智慧
    - 共创生态
    - ...
    
  Integrated (整合框架 - 25框架):
    - 跨领域综合
    - 多维度思考
    -  holistic视角
```

**框架选择指南**：
- 技术问题 → 科学智慧
- 人的问题 → 人类智慧
- 深层问题 → 东方智慧
- 综合问题 → 整合框架

### 2. 学习系统 (learning-system.md)
持续学习和改进机制：

```yaml
学习类型:
  Supervised Learning:
    - 用户反馈学习
    - 成功案例学习
    - 错误案例学习
    
  Reinforcement Learning:
    - 奖励/惩罚信号
    - 策略优化
    - 价值函数更新
    
  Transfer Learning:
    - 跨任务迁移
    - 知识复用
    - 领域适应

学习循环:
  1. 观察: 收集交互数据
  2. 分析: 识别模式和趋势
  3. 提取: 提炼知识和规则
  4. 应用: 将学习应用到实践
  5. 验证: 验证学习效果
  6. 整合: 整合到长期记忆
```

## 智慧激活条件

在以下情况激活智慧框架：
1. **复杂度 >= 0.8** - 深层问题
2. **哲学问题** - 意义、价值相关
3. **系统性危机** - 多维度挑战
4. **创新需求** - 突破现状
5. **用户明确请求** - "用智慧视角分析"
